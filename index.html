☁️ Setup Google Sheets
Tentang Fitur Cloud: Data Dimsum28 akan disimpan permanen di Google Sheets milik Anda. Gratis, aman, dan bisa diakses dari mana saja.
Paste URL Web App dari Google Apps Script di sini
📋 Panduan Setup (5 Langkah)
1
Buka Google Sheets baru
Buka sheets.google.com → Buat spreadsheet baru → beri nama "Dimsum28 Database"
2
Buka Apps Script
Di Google Sheets: klik menu Extensions → Apps Script
3
Paste kode script
Hapus semua kode yang ada, paste isi file google-apps-script.js dari paket download. Simpan (Ctrl+S).
4
Deploy sebagai Web App
Klik Deploy → New deployment
• Type: Web app
• Execute as: Me
• Who has access: Anyone
Klik Deploy → izinkan akses → salin URL
5
Paste URL di sini & Setup Data
Paste URL ke kolom di atas → klik Simpan & Hubungkan → klik ☁️ Simpan ke Cloud di bar atas untuk upload data awal ke Google Sheets.
⚠️ Penting: Setiap kali ada perubahan data, klik ☁️ Simpan ke Cloud agar tersimpan. Gunakan 🔄 Muat dari Cloud untuk sinkronisasi data terbaru di perangkat lain.
